FAQ
  1. What's this?

    Let's say this is a "Data Processing Pipelines"

  2. What's Pipeline?

    The Pipeline is a list of "Processors" which Transforms Data in the Way You Determined.

    Example:

    • Upload an Image
    • Resize to 200x200
    • Grayscale it
  3. How do I Try Pipeline?

    Navigate to "Projects", select a Project, then a choose Pipeline.

    Let's assume that Your Pipeline has ID 1234

    To Upload an Image we would need to execute this request 

                            
curl -X POST -F "file=@/full/path/to/file.png" \
    "http://127.0.0.1:8000/api/v1/free/pipelines/1234/process/"
    And receive a Result
